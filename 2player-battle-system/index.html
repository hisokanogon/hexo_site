戰鬥中顯示血量

如果只是要顯示文字的話可以使用這樣的代碼:

  • text = 文字
  • [ena_hp] = 敵人現在的血量
  • [enaMax_hp] = 敵人滿血的血量
  • xpos 0.1 ypos 0.2 –> 文字出現的位置

或許你會想要在敵人血量歸 0 時讓血量條消失:

  • if ena_hp > 0: –> 如果敵人1的血量高於 0
    • text “[ena_hp]/[enaMax_hp]” xpos 0.1 ypos 0.2 –> 顯示文字
  • else: –> 如果敵人血量小於 0
    • pass –> 甚麼也不做

又或許你想要根據敵人的血量改變文字顯示的顏色
以下是顯示 1 位敵人血量的範例:

  • screen hpStats –> 新的一個叫做 hpStats 的 screen
  • if ena_hp > 0: –> 如果敵人1的血量高於 0
    • if ena_hp > 700: –> 如果敵人的血量高於 700 的話顯示
      • text “[ena_hp]/[enaMax_hp]” xpos 0.1 ypos 0.2 –> 普通的樣子
    • elif ena_hp > 300: –> 如果上一句不成立 + 敵人血量高於 300 的話
      • text “{color=#FFA500}[ena_hp]/[enaMax_hp]{/color}” xpos 0.1 ypos 0.2 –> 顯示橘色文字
    • else: –> 如果上面 2 句不成立 = 血量低於 300 的話
      • text “{color=#ff0000}[ena_hp]/[enaMax_hp]{/color}” xpos 0.1 ypos 0.2 –> 顯示紅色文字

第二位敵人或是主角團的血量條可以就用上面的代碼顯示，改掉變量名字就可以了

戰鬥系統

戰鬥的話在這個教學 / 範例中差不多會以下面的形式呈現

  • 只要2個敵人的血量沒有歸 0 就會一直持續下去
  • 如果主角團 2 個人的血量都歸 0 的話就會直接進入 BE
  • 每個回合會總是以主角團開始攻擊
  • 主角團的人攻擊結束後才會輪到敵人
  • 敵人受到的傷害 = 玩家的戰鬥力 (範例: cha_att) - 敵人的防禦力 (範例: ena_def)
  • 敵人會優先攻擊角色 A –> 角色 A 死亡後才會攻擊角色 B

那首先，先讓我們完成 1 和 2 吧

  • while ena_hp > 0 or enb_hp > 0: –> 當敵人 A 或是 B 血量大於 0 時 (只要其中一個敵人符合條件就可以)
    • if cha_hp > 0 and chb_hp > 0: –> 如果角色A 和 B 血量大於 0 時 (2個角色都要符合條件)

現在讓角色 A (/ 角色 1) 攻擊吧
(角色 B (/ 角色 2)也可以直接使用以下的代碼，只是要把變量改變)
我們可以選擇讓玩家可以自由決定要攻擊哪個敵人
(當然如果你不想讓玩家選擇攻擊的敵人的話就不需要使用以下的代碼 –> 可以直接 call 畫面就好了)

  • Cha 1 的回合! 請選擇攻擊對象! –> 玩家能看到的文字

  • if ena_hp > 0 and enb_hp > 0: –> 如果敵人 1 和 2 血量都大於 0

    • menu: –> 出現選像
      • “敵人1”: –> 玩家可以選擇攻擊敵人1
        • call cha1_atta_en1 –> 如果選擇攻擊敵人1那就進入攻擊敵人1的 label
      • “敵人2”: –> 或是選擇攻擊敵人2

普通攻擊

如果玩家選擇使用普通攻擊的話，你可以使用類似以下的代碼:

  • ena_hp –> 敵人的血量
  • -= –> 減掉的量 (-) / 等於多少 (=)
  • cha_att - ena_def –> 減到的量等於 -> cha_att (主角A的攻擊力) - (減掉) ena_def (敵人A的防禦力)

當然你也可以使用其他變量的組合來執行敵人減掉的血量

顯示被攻擊的特效

接下來或許你會想要顯示特效 (可能敵人流血的特效或是技能打出去的特效)

以上的範例是用來顯示敵人流血

  • show blood –> 秀出流血的圖片

  • at battle3 –> 在之前定義過的 battle3 的位置上

  • with Dissolve(.25) –> 使用 Dissolve 這個效果出現

  • pause .75 –> 圖片將停留 0.75 秒

  • hide blood –> 之後消失

之後你可能還會希望敵人在血量歸 0 後消失或者更換立繪的話就可以使用類似以下的代碼:

  • if ena_hp <= 0: –> 如果敵人1的血量低於或是等於 0 的話
    • hide en1 –> 隱藏敵人1 的立繪 (可以把這行換成替換圖片的代碼窩)
  • if ena_hp <= 0 and enb_hp <= 0: –> 如果 2 位敵人都陣亡了的話

技能攻擊

另外一個選項是技能攻擊，你可以給玩家設定一個 mp 值 (能量 / 藍量的感覺)
或是限制玩家能使用技能的次數
如果要設定技能次數的話你可以事先建立類似以下的變量

  • $ cha_skill = 1 –> 角色A 的技能量 (能使用的技能次數) = 1
  • $ chb_skill = 1 –> 角色B 的技能量 (能使用的技能次數) = 1
    你可以在使用技能後減 1 那玩家就不能再次使用技能了

或是你也可以建立這樣的變量

  • $ cha_mp = 100 –> 角色A 的 mp 量 = 100
  • $ chb_mp = 100 –> 角色B 的 mp 量 = 100

然後你可以設定每次使用技能要減掉多少 mp 量

技能攻擊除此之外跟普通攻擊的代碼都差不多

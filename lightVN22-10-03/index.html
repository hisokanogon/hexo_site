Light.VN 更新紀錄 |【2022-10-03更新】

發表於2022-10-29|更新於2022-12-11|game_dev遊戲開發遊戲引擎lightVN
|字數總計:415|閱讀時長:1分鐘|閱讀量:

版本: 13.4.1

更新重點整理:

  • 實時預覽輕量化
  • 使用 markdown 語法
  • 新增使用 2D 素材構建 3D 地圖小遊戲的範例
  • 可自由擴展相機
  • 在 "LightTest" 中顯示引擎版本
  • 顯示/反映更改語言時的改變
  • 可以在按鍵觸發選項中看到按鍵的狀態
  • 在各個選項卡上顯示螢幕區域信息
  • 增加各種 Function (功能)
  • 旋轉圖片現在可以受暫停命令影響
  • 計算公式時顯示更多錯誤信息
  • 夠在系統消息等命令中使用\n (換行符號 / 提示 - Renpy 也是使用 \n 當作換行符號)
  • 增加使用圖像命令檢查圖像大小是否相同
  • 添加命令 -> 刪除全局信息
  • 添加命令 -> 添加圓形碰撞範圍
  • 添加命令 ->  3D 追蹤相機
  • 添加命令 -> 文字宏 (macro)
  • 添加命令 -> 系統對話框

優化:

  • 實時預覽輕量化
  • 部分錯誤發生時使用更明顯的音效
  • 提高同步聲音的準確性

錯誤/功能修正:

  • 由於 tilemap 導致強制關閉程式的問題
  • 實時預覽和 csv 的問題
  • 碰撞優先級的問題
  • 圖像過場產生的問題
  • 跳過指令相關的問題
  • 實時預覽時的“重複執行”的問題

  • 檔案大小為: 563 MB

  • 目前使用 Light.VN 製作的視覺小說數量: 146 

  • 2022 至今為次以推出了 14 個作品

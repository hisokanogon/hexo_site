NVL Maker | 介紹 & 使用

發表於2022-10-30|更新於2022-12-11|game_dev遊戲開發遊戲引擎nvl maker
|字數總計:1.3k|閱讀時長:3分鐘|閱讀量:

什麼是 NVL Maker?

一個小小的關於 NVL Maker 的介紹~

  • 他是用來製作視覺小說 / 文字冒險遊戲的遊戲引擎
  • 特點是它的視覺化，使用 NVL Maker 你可以不用打任何代碼就能製作出一個很讚的視覺小說
  • 雖然你不需要代碼就能使用但如果你需要使用代碼的話 NVL Maker 也提供你可以自己打代碼的能力

其他免費的視覺化遊戲引擎還有

  • Belle
  • Novelty
  • Light.VN

下載,安裝與初使用

NVL Maker 提供了繁體/簡體,日文以及英文的翻譯
可以自行選擇喜翻的語言

  • 下載可以透過這裡 下載NVL Maker

你可以透過 Google, 百度或是 Bitbucket 下載檔案
解壓縮 ZIP 檔案後你可以點擊一下 Editor 打開 NVL Maker 的程序檔

有些人可能會出現打不開或是出現亂碼的情況，如果你也發生了這樣的情況的話你可以試試看

  • 關掉防火牆
  • 改變你的地區和時間 (都改成台灣就對了)

如果你是用 Win 11 的話可以從設定的左排工具列進入時間和語言

然後更改你的地區和時間, 還有系統語言 (改成中文, 如果你原本就是用中文的話就不用更改拉)

創建新專案

要創建新的專案的話你可以透過點擊上排工具列最左邊的圖標創建

由一些主題可以選擇，你可以看你比較喜翻哪一個

改變主題默認的分辨率會造成圖片大小和按鈕位置對不上

當你創建好新的專案後就可以開始製作你的遊戲拉~

使用 NVL Maker 製作遊戲吧!

編輯腳本

當你創建好專案後想要開始編輯劇情的話可以到腳本編輯 打開一個已經存在的腳本或是建立新的腳本
一個腳本大概長以下這樣:

要新增腳本在其中一行的話可以透過 單行操作 下的 + 號插入一行腳本
或是透過 - 號刪除一行

覺得一行一行編輯很麻煩的話你也可以過直接編輯來編輯腳本

點一下直接編輯後你就能看到以下的畫面拉

如果你的電腦上沒有任何程式編輯器的話，建議你先下載一個
一些常見的程式編輯器:

  • Visual Studio
  • Atom
  • Notepad++

使用指令

重點來拉，使用 NVL Maker 是不需要打任何代碼搭，只要使用指令就可以拉
有哪些指令你可以通過下圖看到

一個可能會讓各位小夥伴們 (我w) 有點困惑的地方

  • 如何離開腳本?

上面的問題可能是有些人會遇到的
其實要離開 / 關閉腳本是很簡單搭只要好好看左上方的按鈕就可以發現一個叫做 關閉腳本 的按鈕
點一下就可以離開拉~

特色

接下來稍微介紹一下 NVL Maker 的一些特色

  1. 不用代碼就能製作遊戲
  2. 更換 UI 可以只用滑鼠拖曳就完成
  3. 很多已經做好的UI畫面，如果你對他們滿意的話就可以直接開始製作你的故事
  4. 只需要點擊指令就可以完成想要執行的動作了
  5. 可以輕鬆的製作地圖
  6. 有中文的畫面可以使用

NVL Maker 相似的引擎

如果你覺得 NVL Maker 你用不習慣或是出於其他原因在尋找其他引擎的話
以下是一些同樣是視覺化然後使用方式也是放上指令就好的引擎:

  • TyranoBuilder
    雖然是付費引擎只是打折時只要 8 美元(199元台幣) 沒打折時也只要 15 美元(398元台幣)
    本身不是很貴算是很能負擔的價格而且只要一次付費後就能一直使用了
    關於 TyranoBuilder 的更多介紹和評價我在原網站都寫過了所以這裡就先不寫拉

  • Light.VN
    個人以引擎外觀，功能等來看都很喜歡
    唯一一個缺點是它只有日文的介面，因為我日文完全不行所以用不了
    希望哪天哪個大人可以漢化它

Online (非桌面程式) 的引擎的話

  • 易次元
    我個人蠻推薦易次元的 !!(但我個人沒用主要是覺得之後可能會需要實名認證 - 就像中國的其他網站那樣)!!

  • CloudNovel
    可以讓你線上製作視覺小說的網站，主要用戶為歐美地區
    有一些限制 (不管是作為創作者還是用戶) 但可以讓你免費發布和製作遊戲

  • 非獨家委託有可能會上架 Itch.io

委託流程 ★ Commission Process

❖ 寄送委託 ❖

  • 偏好使用 Email | 聯絡信箱

委託郵件的格式請參考以下的格式 (在價格區 - 請繼續往下滑)

⬇️

❖ 具體討論 ❖

  • 收到信後，我會盡快回信給你
  • 之後可以再做更進一步的討論
  • 像是最後的收費, 具體的項目, 雙方接受 / 不接受的範圍等…
  • 合約內容也會在這裡談好

⬇️

❖ 確認委託 / 簽訂合約 ❖

  • 確認委託後我會先要求 50% 的訂金
  • 收到訂金後我才會開始製作窩
  • 每次收款會額外收 $50 作為手續費 (通常轉到我的帳號時會扣掉4-6%的手續費 QAQ)

※ 如果你信任我的話也可以一次付清 (❁´◡`❁)ww

  • 一次付清的話每次更新寄給你的檔案就會不含浮水印拉
  • 在設計 GUI 方面我是不太建議，不確定做出來的 UI 符不符合你的期待
  • 在 UI 代碼方面個人偏好一次付清
  • 付款方式偏好 PayPal / Stripe, 沒有辦法的話也接受 Steam 禮物卡~

⬇️

❖ 正式製作 ❖

  • 在這期間會定時將完成的部分通知給你
  • 同時包含代碼製作的話會在確認完所有畫面後在開始製作代碼
  • 整體專案會在收到 100% 約定金額後一次寄送給你

⬇️

❖ 結束 ❖

  • 在送出專案給你 + 獲得確認後就會當作結束此委託

委託價格 ★ Pricing

  • 價格的話會考量到「非商業/非商業」,「獨家/非獨家」, 項目規模等因素有所差別
  • 具體的價格可以在郵件中詳細討論
    ➤ 未特別註明的話價格均為新台幣(NTD)。

➤ 會員享有額外特價，會在談價時注意

➤ 代碼委託的話請提供素材

​

以下的價格都可以透過溝通討論在確定最終價格窩

【常駐價格 | 委託單一畫面時用】

    1. 封面
    2. Save

    4. 介紹 (About)
    5. 對話/選像框
    6. 設定畫面

  • 進階畫面: $560

郵件格式:

  • 以下的格式可以一鍵複製窩, 如果沒出現圖案的話可以刷新頁面確認

【組合包】

  • 基礎組合包: $1300 (含封面, 設定, 存檔/讀檔, 介紹, 歷史紀錄, 對話畫面)
  • 進階組合包: $2000 (含基礎包, 遊戲選單, 製作人員畫面, 快速選單, 通知畫面 (Confirmation))

【常駐價格 | 委託單一畫面時用】

  • 基礎畫面: $250+
    1. 封面
    2. Save
    3. Load
    4. 介紹 (About)
    5. 對話/選像框
    6. 設定畫面
    7. 快速選單
    8. 通知 (Confirmation Screen)
  • 進階畫面: $350+
    1. 相簿
    2. 遊戲選單 (Navigation)
    3. 角色/好感畫面
    4. 製作人員列表
    5. 對話回顧 (History)
    6. 音樂房間
  • 額外項目:
    1. 地圖畫面 ➡️ 底價: $500
      • 默認自帶 5 個互動元素, 超過需額外加價 = 50/互動元素
    2. 自動跳出好感訊息 ➡️ $150
    3. 更換滑鼠造型 ➡️ $50
    4. 根據章節/結局更換封面 ➡️ $100
    5. 協助劇情樹或時間線整理/製作 ➡️ $ 120
    6. 額外代碼解釋 / 教學 ➡️ $ 450
      • 這裡會製作完整的 PPT / PDF -> 會一次發送給你/自動更新因為需要較長時間製作所以價格比較高
      • 不需要PPT/PDF 偏好直接透過社群軟體或是Email協助的話也可以 ➡️ $100+ 主要看專案大小
    7. 試玩 / 測試遊戲 / 評價 ➡️ $100+ 根據遊戲長度
    8. 其他 (所有你能在這個網站中找到的教學都可以委託我製作)

接受範圍:

  • 基礎 Renpy 功能
  • 直線劇本
  • 多線劇本
  • 好感系統
  • 多結局
  • 地圖等和
  • 其他你能在這個網站中找到的教學

價格的話根據難度而定所以關於遊戲代碼的委託請詳細說明你的專案

  • 郵件格式請參考前一個項目

接受範圍:

  • 代碼腳本 & GUI委託

接受的範圍已經寫過了所以這裡也來一個不接受的範圍

【不接受】

  • 複雜的戰鬥系統
  • 商城
  • 3D
  • R18+ 不接受聲音 (你可以自己加入背景音樂/配音)
    • 製作時被家人/室友聽到會很尷尬所以為了避免這個情況暫時不接受

  • 以下有幾點會影響到價格

【關於授權】

  • 希望可以憑良心遵守
  • 商用/獨家委託的價格也不會高出多少倍
  • 如果要用在商用專案上，請讓我知道
  • 如果預算真的不允許是可以談的

  • 可再使用/非買斷: 接受免費範圍再次使用 - 作為教學或是影片素材等但不希望再次販售

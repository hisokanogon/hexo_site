在開始之前如果你還不知道視覺小說是什麼或是不知道為什麼要用其他小眾的平台先讓我來介紹一下視覺小說吧
(不要在問我為什麼不用/不學 Unity 之類的問題好嗎)

視覺小說遊戲到底是在玩什麼?

  • 視覺小說 / Visual Novel 是一種以文字和圖像為主要表現手段的互動故事遊戲，它們通常有多種選擇和分支，讓玩家可以影響故事的走向和結局
  • 它也是日本的一種流行文化產物，也有越來越多的國外創作者和玩家對這類型的遊戲感興趣。
  • 視覺小說遊戲的類型和風格可以非常多樣，有些是輕鬆幽默的日常喜劇，有些是驚悚刺激的推理懸疑，有些是浪漫感人的愛情故事，有些是奇幻冒險的英雄傳奇，等等
  • 不同的視覺小說遊戲也有不同的目標群體，有些是專為女性或男性設計的，有些是適合全年齡或限制級的，有些是主打劇情或遊戲性的, etc.
  • 玩家可以根據自己的喜好和需求，選擇合適的視覺小說遊戲來玩

  • 除了可以帶給玩家娛樂和放鬆，也可以啟發玩家的想像力和創造力，甚至可以讓玩家學習到一些知識和技能

    • 例如，有些視覺小說遊戲涉及到歷史、文化、科學、哲學等主題，可以讓玩家增廣見聞
    • 有些視覺小說遊戲包含了解謎、策略、模擬等元素，可以讓玩家鍛煉思維和判斷；有些視覺小說遊戲探討了人性、社會、倫理等問題，能讓玩家反思自己和世界

  • 如果你想要製作自己的視覺小說遊戲，那麼你會需要選擇一個適合的遊戲引擎

  • 遊戲引擎是一種軟件，它提供了遊戲開發所需的各種功能和工具，時常專注在視覺小說的引擎還會自帶多種做好的範本可以直接使用也加快了遊戲製作的流程，讓開發者可以更容易地設計和實現自己的想法

Suika 2：一個開源、跨平台、原生的視覺小說引擎

現在進入到這篇文章的重點!

在這篇文章中，我會介紹一個名為 Suika 2 的視覺小說遊戲引擎，它有以下的特點：

  • 開源：Suika 2 的原始碼是完全公開的，任何人都可以免費使用、修改和分發它，而不用擔心版權或授權問題。Suika 2 採用了 MIT 協議，這是一種非常寬鬆的開源協議，它允許商業和非商業的使用
  • 跨平台：Suika 2 可以在多種平台上運行，包括 Windows、Mac、iOS 和 Android。你可以用 Suika 2 製作出可以在不同設備上玩的遊戲，而不用擔心兼容性或轉換問題。Suika 2 還支持 Web 瀏覽器和 Nintendo Switch 的非官方移植
  • 原生：Suika 2 是用 C 語言編寫的，它不依賴於 Web 或 Python 等其他技術。這意味著 Suika 2 可以直接運行在硬件上，而不需要任何中間層或解釋器。這可以提高遊戲的性能和穩定性，並減少資源的消耗

除了這些基本特點外，Suika 2 還有以下的優勢：

  • 簡單易用：Suika 2 提供了輕量而完整的開發工具（Suika 2 Pro），它可以幫助你快速地創建和測試你的遊戲。你可以用滑鼠操作來設計場景、角色、對話和選項，也可以用簡單的指令來控制遊戲邏輯和效果
  • 輕鬆分發: Suika 2 Pro 還有一鍵導出功能，你可以指定目標平台並導出你的遊戲。例如，你可以導出 Windows 的安裝包、Web 的數據或 Android/iOS 的原始碼
  • 輕便靈活：它適用於各種類型和風格的視覺小說遊戲，無論是女性向、男性向還是全年齡向，無論是短篇還是長篇，無論是單線還是多線，無論是靜態還是動態，你都可以用 Suika 2 來實現你的創意。Suika 2 還支持多語言和 Steam 分發，讓你的遊戲可以觸及更多的受眾

現在我們可以從官網下載不同版本的 Suika

Suika2 Pro

編輯器外觀:

點擊編輯器的左上角的按鈕可以開始預覽遊戲

要繼續寫劇本的時候可以按一下 “Pause” 來停止預覽遊戲只是要注意的是它會一直播到當前的句子播放結束

點擊中間範圍的按鈕後就會在右邊出現類似以下的編輯畫面:

右下的透明範圍的地方可以讓你把不要的幾行拖過去然後就會刪掉那行了
或是如果你想預覽專案裡面的圖片的話可以點一下檔案的名字然後圖片預覽就會出現在右下的格子裡面了

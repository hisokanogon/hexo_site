什麼是 Visual Novel Maker / 視覺小說製作大師?

  • Visual Novel Maker 是一款由 Degica 在 2017 年發布的視覺小說製作引擎
  • 跟 Rpg Maker (RPG 製作大師系列) 一樣都是為了方便有遊戲開發夢想卻對代碼不太熟悉的人們製作的遊戲引擎
  • 自從發布以來已經也有一些透過 VN Maker 製作的遊戲發布了有興趣的人可以通過以下的傳送門查看
  • 傳送門1 | 傳送門2
  • 已它是 2017 就已經發布的遊戲引擎來說做出來的作品老實說真的不多，為什麼會這樣會在之後提到搭~
  • !!(個人真的覺得這個引擎很可惜，蠻有潛力的說但…)!!

缺點 / bug

以下我會列出一些讓我不會考慮 / 選擇 VN Maker 的幾個點

  1. 製作 UI 時中間的畫面會一直閃爍 (超級不舒服幫眼睛qq)
  2. 加入人物表情 / 素材很麻煩
  3. 指令很不直觀，對新手很不友善
  4. 教學寫的不是很清楚
  5. 直令疊在一起在往上往下滑的時候容易錯過要找的那行
  6. 無法一次複製多個指令 (如果要換位置到其他行需要一個一個移動)
  7. 不管是指令的使用還是變量的使用 VN Maker 做的都很讓人感到困惑

我相信很多會選擇這些視覺化遊戲引擎的人們都是因為對代碼不是很熟悉
或是其他引擎的 Documentary 看不太懂所以才會來找這些比較簡單好上手的引擎的
但 VN Maker 作為一個付費引擎，價格又設的那麼高真的做的很失敗
引擎畫面超讓人困惑，很多指令點了又點也都不知道要怎麼正確使用
官方的教學也是根本讓人難以看懂
一堆基本的功能都沒教學
用的人又少根本找不到太多民間的教學
影片也是也沒有多少人做
之前我本來是想來做做看但因為真的看不懂官方的教學
自己又摸索不出來就放棄了

簡單一句話 Visual Novel Maker 真的超難用

欸? 好像越寫越氣惹w有真心覺得浪費錢了w
